Get Started

Use a public RPC node, and get started on the Crystaleum network within seconds! Or build from source and run geth to sync the chain ID 103090 which is initialized in official crystaleum EVM core. The Chain ID is 103090 listed on official chainlist. RPC node is listed on official Ethereum chain list Native integrations are already available for MetaMask, Infura, Alchemy, and beyond!!! What is Crystaleum?

@Crystaleum, an alt-ethereum segregated from main ethereum, conducting research in active development on Binance Smart Chain, Polygon, Ethereum, Cronos and more alt-ethereum EVM compatible network. Partnerships are being formed with decentralized exchanges, DeFi protocols, and more exciting news regarding these and forthcoming deployments to be announced soon! @crfinews
